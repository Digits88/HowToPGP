Installation des Mailbelope-Plugins

Als nächstes musst du das Mailvelope-Plugin für deinen Browser installieren. Hier kannst du das Plugin für deinen Browser herunterladen. Klicke bei beiden Sicherheitsfragen auf "Akzeptieren" bzw. "Installieren".
Nach der Insallation kannst du das Mailvelope Hauptmenü über, das in deiner Browsertoolbar erschienene Sicherheitsschlossicon erreichen.
